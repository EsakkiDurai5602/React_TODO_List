# Todo List App

A simple and responsive Todo List application built using React. This app helps users manage daily tasks efficiently with features like adding, editing, deleting, and marking tasks as completed.

## Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Store tasks using local storage
- Responsive user interface
- Simple and user-friendly design

## Technologies Used

- React
- JavaScript
- HTML
- CSS
- Local Storage
